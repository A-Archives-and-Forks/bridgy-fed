IndieWeb user instructions:

  1. Redirect all /.well-known/* URLs from your site to https://fed.brid.gy/.well-known/* with HTTP 301 or 302.
    • If you're on WordPress, you can use a plugin like Safe Redirect Manager.
    • If you're on Apache, add this to your .htaccess:
      RewriteEngine on
    RewriteBase /
    RewriteRule ^.well-known/.* https://fed.brid.gy/$0  [redirect=302,last]
    • If you're on nginx, add this to nginx.conf in your site's server section:
      rewrite ^/.well-known/ https://fed.brid.gy/$request_uri redirect;
  2. If your site doesn't already have an Atom feed, add one.
    • If you're on WordPress, install the Atom Default Feed plugin.
    • Otherwise, you can use granary. Just add this to your site's HTML <head> and fill in YOUR DOMAIN with your site's domain:
      <link rel="alternate" type="application/atom+xml" href="https://granary-demo.appspot.com/url?url=http://[YOUR DOMAIN]/&input=html&output=atom&hub=https://bridgy-fed.superfeedr.com/" />
